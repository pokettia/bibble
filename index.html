Query SPARQL – Notte Stellata (Vincent van Gogh)

Utilizzando Wikidata e le informazioni collegate a Notte stellata, è possibile interrogare il knowledge graph per recuperare dati sull'opera, l'autore e immagini associate.

SELECT DISTINCT ?painting ?paintingLabel ?creatorLabel ?image WHERE {
  ?painting wdt:P31 wd:Q3305213 ;       # Istanza di dipinto
            rdfs:label "The Starry Night"@en ;
            wdt:P170 ?creator ;         # Autore dell'opera
            wdt:P18 ?image .            # Immagine dell'opera

  SERVICE wikibase:label { bd:serviceParam wikibase:language "it,en". }
}
LIMIT 1

Risultato atteso:

  • Opera: Notte stellata
  • Autore: Vincent van Gogh
